# Perfumería React

Proyecto final del curso React JS – Coderhouse.

Aplicación e-commerce desarrollada con React que utiliza Firebase Firestore para la gestión de productos y órdenes.

## Tecnologías
- React
- Firebase
- React Router
- Context API

## Funcionalidades
- Listado de productos desde Firestore
- Carrito de compras
- Eliminación de productos del carrito
- Checkout
